## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/SaniyaSheldarkar/neuroFleet.git
cd neuroFleet
```

---

### 2. Backend Setup

```
cd backend
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
uvicorn main:app --reload
```

---

### 3. Frontend Setup

```
cd frontend
npm install
npm start
```
